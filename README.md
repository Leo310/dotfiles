# Installation
### Download and change folder
```
cd ~
git clone https://github.com/Leo310/dotfiles.git

cd dotfiles
```
### Run install (using stow)
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller (same as installer)
```
./install.sh
```
